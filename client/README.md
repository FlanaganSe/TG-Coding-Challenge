# TG Exercise Front End

Built with React + Typescript. Project was bootstrapped with Vite.

#### Running the client

```zsh
npm i
npm run dev
```

### Running tests

```
# Run all tests
npm run test

# Run vitest ui
npm run test:ui
```

#### Client side libraries

- Zustand
- Axios
- Tailwindcss
- Vitest
